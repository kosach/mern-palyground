# Description
MERN playground;

## Start in Docker for Dev
Install docker & docker-compose
Start app:
```
docker-compose -f docker-compose.dev.yaml up
```
# DB config
```
./config/default.json
```
App url: http://localhost:3000

# DB env:
```
DB_USERNAME
DB_PASSWORD
DB_HOST
DB_DATABASE
DB_PORT 
```
## Start local
```
cd ./client
npm install
cd ..
npm install
npm run dev

```
